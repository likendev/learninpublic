# Arrays

## Static Arrays
- Static arrays have an allocated size when initialized.
- E.g.: Java, C++, C etc
- |Operations	|Time Complexity|Notes|
  |-------------|---------------|-----------|
  |Reading		|	O(1)		|\-|
  |Insertion	|	O(1)		|If inserting at the end, O(1)|
  |Deletion		|	O(n)		|If deleting at the end, O(1)|
